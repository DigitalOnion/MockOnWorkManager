Mock On WorkManager

This sample app shows how to create a test case for a Worker object injecting a fake dependency.

The app is a very simple "Type and Reverse" app. For the purpose of the exercise, it uses a TextReverser to act as a text reverse server.

I use a Worker to schedule a WorkManager task to call the text reverse server, and I inject the TextReverser object into the Worker task.

The challenge is to create a test case which will inject a Fake Text Reverse Server.

It works, check it out.
